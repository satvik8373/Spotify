📝 What This Does

  2. Removes them so fresh data can be fetched
  3. Reloads the app to show the new trending playlist

✅ After Clearing Cache

  1. Make sure backend is running on port 5000
  2. Make sure frontend is running
  3. Go to the home page
  4. Look for "🔥 Trending Now - Mavrixfy" as the FIRST playlist!
