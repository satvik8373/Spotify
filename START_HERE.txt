  HOW TO ACCESS THE NEW JIOSAAVN FEATURE (50+ SONGS, NOT 10!)
═══════════════════════════════════════════════════════════════

STEP 1: Start Your Servers
---------------------------
Terminal 1 (Backend):
  cd "Mavrixfy - Copy/backend"
  npm start

Terminal 2 (Frontend):
  cd "Mavrixfy - Copy/frontend"
  npm run dev


STEP 2: Open the Feature
-------------------------
Open your browser and go to:

  http://localhost:5173/jiosaavn/browse

That's it! You'll see the new song browser.


WHAT YOU'LL SEE:
----------------
✅ 40-50 songs per category (not just 10!)
✅ Search for any song/artist
✅ Browse by category (Bollywood, Romantic, Party, etc.)
✅ Select multiple songs
✅ Add them to your playlists


QUICK TEST:
-----------
1. Go to: http://localhost:5173/jiosaavn/browse
2. Click "Start Browsing"
3. Click "Bollywood" tab
4. See 40-50 songs displayed!
5. Click on songs to select them
6. Click "Add X Songs" to add to playlist


PROOF IT WORKS:
---------------
Backend API test (paste in browser):
  http://localhost:5000/api/jiosaavn-songs/search?query=bollywood&limit=50

You'll see 50 songs in JSON format!
