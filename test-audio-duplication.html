Audio Player Duplication Test

Open the browser console and navigate to your app. Then run this in the console:

// Check how many audio elements exist
console.log('Number of audio elements:', document.querySelectorAll('audio').length);

// Check how many player controls exist
console.log('Desktop players:', document.querySelectorAll('[class*="fixed bottom-0"]').length);
console.log('Mobile players:', document.querySelectorAll('[class*="fixed bottom-14"]').length);

Expected results:

  • Number of audio elements: 1
  • Desktop players: 1 (only PlaybackControls)
  • Mobile players: 1 (only PlaybackControls)
